BigCloneEval Web Framework

Its a web framework of BigCLoneEval Framework. Please fill up the following field to evaluate your clone detection tool woth respect to BigCloneBench Oracle. Thanks

Please upload the clone result in the format of BigCloneBench. You can take help from khaledkucse/bigcloneeval:detection:latest image from docker hub.

How reference clone similarity should be measured: (both/line/token/avg). Default both.

Specify the clone matcher. See documentation for configuration strings. Default is coverage-matcher with 70% coverage threshold.

Minimum clone similarity to evaluate for. In range [0,100], must be a multiple of 5.

Adds a minimum clone size, in lines, constraint to the reference clones.

Adds a maximum clone size, in lines, constraint to the reference clones.

Adds a minimum clone size, in pretty-printed lines, constraint to the reference clones.

Adds a maximum clone size, in pretty-printed lines, constraint to the reference clones.

Adds a minimum clone size, in tokens, constraint to the reference clones.

Adds a maximum clone size, in tokens, constraint to the reference clones.

Adds a minimum judges constraint to the reference clones.

Adds a minimum confidence constraint to the reference clones.
